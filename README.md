# Численное решение уравнений газовой динамики

## Описание проекта

Программа для численного решения системы уравнений газовой динамики методом конечных разностей с использованием метода прогонки (трёхдиагональный алгоритм).

### Решаемая система уравнений

Программа решает систему:
- Уравнение неразрывности для G = ln(ρ)
- Уравнение движения для скорости V

```
evm_project/
```

## Параметры расчёта

Программа выполняет два типа тестов:

### 1. Тесты на различных сетках
- Размеры сеток: 10, 100 узлов
- Режимы: mode=0 (уравнение состояния p ~ ρ^γ)
- Параметры газа: C_rho = 1, mu = 0.1

### 2. Тесты на сходимость
- Измельчение сетки с коэффициентами 2, 4
- Вычисление невязок в нормах C, L2, W2

## Результаты

Результаты сохраняются в директории `tex_logs/` в формате LaTeX таблиц:
- `mu_X_C_rho_Y_modeZ_G.tex` - результаты для решения G (логарифм плотности)
- `mu_X_C_rho_Y_modeZ_V.tex` - результаты для решения V (скорость)

Каждая таблица содержит невязки в трёх нормах:
- **C** - равномерная норма (максимум модуля)
- **L2** - интегральная норма
- **W2** - норма в пространстве Соболева

## Численный метод

### Метод прогонки
Используется для решения системы линейных уравнений с трёхдиагональной матрицей:
1. Прямой ход - вычисление прогоночных коэффициентов
2. Обратный ход - вычисление решения

### Схема расщепления
На каждом временном слое последовательно решаются:
1. Уравнение для G (логарифм плотности)
2. Уравнение для V (скорость)

## Граничные условия

- На левой границе (x=0): условия первого рода с односторонней разностной аппроксимацией
- На правой границе (x=1): условия первого рода с односторонней разностной аппроксимацией
- Для скорости V: нулевые граничные условия (V=0 на границах)

## Тестовые функции

Программа использует следующие аналитические решения для проверки точности:

```
ρ(x,t) = exp(t) * (cos(3πx) + 1.5)
u(x,t) = cos(2πt) * sin(4πx)
```

## Вывод программы

Программа выводит:
1. Прогресс вычислений для каждой комбинации параметров
2. Результаты анализа сходимости при измельчении сетки
3. Невязки численного решения относительно точного решения

## Примечания

- Для больших сеток (1000, 10000 узлов) расчёт может занять значительное время
- В упрощённой версии используются только сетки 10 и 100 для быстрого тестирования
- Для полных расчётов раскомментируйте дополнительные параметры в main.cpp
